# Template sources (when user does NOT provide a template)

> 목적: "템플릿 미제공" 상황에서 신뢰할 수 있는 Word 기반 공문/레터 양식을 신속하게 확보하기 위한 소스 목록.
> 주의: 소스별 라이선스/사용권을 반드시 확인.

## 1) 1순위: 공식/공공 서식 (국가/기관)
### 한국(공문/행정 서식)
- 법제처 법령정보: 정부공문서규정 서식(기안용지 등) — 구조/필드 참고용
  - https://law.go.kr/ (정부공문서규정 서식 페이지 검색)
- 행정안전부 문서24
  - https://docu.gdoc.go.kr/
  - 문서24는 공문작성 서식/웹서식 연계가 있음을 참고

### 공공서식 작성 도구(참고)
- 행정안전부 "공공서식 한글" 배포 안내
- 한컴 "공공 한글" 다운로드/설명
  - (Word 템플릿은 아닐 수 있으나 공공서식 구조/필드 표준을 참고 가능)

## 2) 2순위: 벤더 제공 Word 템플릿(신뢰도 높음)
- Microsoft Create: Letters & letterhead templates
  - https://word.cloud.microsoft/create/en/letters-templates/
- Microsoft Create: Business letters
  - https://word.cloud.microsoft/create/en/business-letters/
- Microsoft Support: "Download free, pre-built templates"
  - https://support.microsoft.com/en-us/office/download-free-pre-built-templates-29f2a18d-29a6-4a07-998b-cfe5ff7ffbbb

## 3) 3순위: 신뢰 가능한 무료 템플릿 라이브러리(가벼운 기본형)
- Vertex42: Business Letter Template for Word (표준 블록 포맷)
  - https://www.vertex42.com/WordTemplates/business-letter.html

## 4) 4순위: 커뮤니티/마켓/템플릿 플랫폼(품질 다양, 라이선스 확인)
### 한국 문서 템플릿 커뮤니티/플랫폼
- 예스폼: 공문/공문양식 검색
  - https://www.yesform.com/
- 비즈폼: 공문 문서양식/예제
  - https://www.bizforms.co.kr/

### 글로벌 템플릿 플랫폼
- Canva: letterhead templates (Word 직출력은 케이스별 상이 — 필요 시 PDF→DOCX 변환 고려)
  - https://www.canva.com/letterheads/templates/
- Template.net: Word letterhead templates (유료 비중 큼)
  - https://www.template.net/letterheads/word

## 5) 국제/관행 기반 레이아웃 가이드("템플릿 생성" 시 근거)
- DIN 5008: 유럽권(특히 독일) 비즈니스 문서 레이아웃의 대표 표준(개념/마진/구성 참고)
- 용지: ISO 216(A4)
- 날짜: ISO 8601

## 6) 추천 검색 쿼리(에이전트용)
- "official letter template docx letterhead"
- "business letter template docx block format"
- "공문서 양식 docx"
- "레터헤드 템플릿 word"
- "정부공문서규정 서식 기안용지"

## 7) 선택 기준(짧게)
- 우선순위: (1) 관할 공식 서식 > (2) Microsoft 템플릿 > (3) 검증된 무료 > (4) 커뮤니티/마켓
- 서식 요소: A4, 여백, 헤더(기관명/로고/주소), 문서번호/시행/수신/제목/본문/첨부/서명 필드 포함 여부
- 라이선스: 상업적 사용/재배포 조건 확인
