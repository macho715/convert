# Outlook 이메일 분석 시스템 v3.0 Lite
## PST 백업 제외 - 용량 절약 버전 ⚡

---

## 🎯 핵심 요약

### ✅ 포함된 기능
- 기간별 이메일 분석 (시작일~종료일)
- 다중 Outlook 폴더 자동 스캔
- 엑셀 리포트 자동 생성 (5개 시트)
- GUI 기반 사용자 인터페이스

### ❌ 제외된 기능
- PST 파일 자동 백업
- SHA256 무결성 검증
- 다중 백업 시스템

### 💡 왜 Lite 버전?
```
✓ 디스크 용량 절약 (PST 백업 불필요)
✓ 빠른 실행 (백업 단계 생략)
✓ 순수 분석 집중
```

---

## 🚀 3단계 빠른 시작

### 1️⃣ 설치 (1분)
```powershell
.\install.ps1
```

### 2️⃣ 실행 (1클릭)
```
run_lite.bat 더블클릭
```

### 3️⃣ 분석 (5클릭)
```
1. [Outlook 연결 및 폴더 스캔]
2. 날짜 범위 설정
3. 폴더 선택
4. [▶ 분석 시작]
5. 완료!
```

## ⚠️ 주의사항

### ✅ DO
- 분석 전 Outlook 정상 작동 확인
- 충분한 시간 확보 (10-20분)
- 필요한 폴더만 선택

### ❌ DON'T
- 분석 중 Outlook 종료 금지
- 분석 중 컴퓨터 끄지 마세요
- 여러 분석 동시 실행 금지

## 🆘 빠른 문제 해결

### 문제: "Python을 찾을 수 없습니다"
```
→ Python 설치: https://www.python.org/downloads/
→ "Add Python to PATH" 체크 필수!
```

### 문제: "Outlook 연결 실패"
```
→ Outlook 수동 실행 확인
→ 관리자 권한으로 실행
```

### 문제: "분석이 느려요"
```
→ 폴더 수 줄이기
→ 기간 범위 축소 (3개월 → 1개월)
```
